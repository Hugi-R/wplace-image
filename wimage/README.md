# wimage
A library to process wplace.live image in my custom format.

## Binaries

- Benchmark zstd compression:
```shell
cargo run -p wimage --bin compression_bench
```

- Benchmark downscale:
```shell
cargo run -p wimage --bin downscale_bench
```

- CLI:
```shell
cargo run -p wimage --bin wimage_cli
```
